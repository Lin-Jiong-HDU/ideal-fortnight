# GEO 优化平台用户指南

本目录包含 GEO（Generative Engine Optimization）优化平台各角色的完整使用指南。

## 指南列表

| 角色 | 文档 | 说明 |
|------|------|------|
| 客户 | [客户使用指南](./customer-guide.md) | 提交内容优化、审核结果、管理配额 |
| 优化师 | [优化师使用指南](./optimizer-guide.md) | 领取工单、执行优化、提交结果 |
| 管理员 | [管理员使用指南](./admin-guide.md) | 用户管理、客户管理、系统配置 |

## 平台简介

GEO 优化平台帮助企业和内容创作者优化内容，使其在 AI 搜索引擎（如 ChatGPT、Perplexity、Google AI、Claude）中获得更好的引用和曝光。

### 角色权限概览

```
┌─────────────────────────────────────────────────────────────┐
│                        管理员 (Admin)                        │
│  全部权限：用户管理、客户管理、项目管理、配额管理、全局工单    │
└─────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
┌─────────────────────────┐     ┌─────────────────────────┐
│     优化师 (Optimizer)   │     │      客户 (Customer)     │
│  • 查看工单池            │     │  • 提交优化工单          │
│  • 领取和处理工单        │     │  • 审核优化结果          │
│  • 执行内容优化          │     │  • 查看配额状态          │
│  • 提交优化结果          │     │  • 查看工单进度          │
└─────────────────────────┘     └─────────────────────────┘
```

### 工单流转流程

```
客户提交 → 待处理 → 优化师领取 → 处理中 → 审核中 → 已完成
                                        │
                                        ▼
                                    客户拒绝
                                        │
                                        ▼
                                     处理中（重新优化）
```

## 快速开始

### 新客户

1. 联系管理员获取账户
2. 登录平台
3. 阅读 [客户使用指南](./customer-guide.md)
4. 开始提交优化工单

### 新优化师

1. 联系管理员获取账户
2. 登录平台
3. 阅读 [优化师使用指南](./optimizer-guide.md)
4. 从工单池领取第一个工单

### 管理员

1. 使用管理员账户登录
2. 阅读 [管理员使用指南](./admin-guide.md)
3. 创建用户和客户
4. 配置项目和配额
